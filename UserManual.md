## 1. บทนำ

### 1.1 วัตถุประสงค์ของคู่มือ

คู่มือฉบับนี้จัดทำขึ้นเพื่อเป็นแนวทางในการใช้งาน **ระบบรายงานผลลัพธ์โดยผู้ป่วย (Patient Report Outcomes Measurement)** สำหรับผู้ป่วยโรคเบาหวานชนิดที่ 2 โดยครอบคลุมทั้งการใช้งานสำหรับผู้ตอบแบบสอบถามและผู้ดูแลระบบ

### 1.2 ขอบเขตของคู่มือ

คู่มือนี้ครอบคลุมการใช้งานระบบในส่วนต่างๆ ดังนี้:

-   การสมัครสมาชิกและเข้าสู่ระบบ
-   การทำแบบสอบถามสำหรับผู้ป่วย
-   การดูรายงานและส่งออกข้อมูลสำหรับผู้ดูแลระบบ

### 1.3 กลุ่มผู้ใช้เป้าหมาย

| ประเภทผู้ใช้               | บทบาท                                                 |
| -------------------------- | ----------------------------------------------------- |
| **ผู้ใช้งานทั่วไป (User)** | ผู้ป่วยหรือผู้ตอบแบบสอบถามที่เข้ามากรอกข้อมูลการวิจัย |
| **ผู้ดูแลระบบ (Admin)**    | เจ้าหน้าที่วิจัยที่ดูแลและวิเคราะห์ข้อมูลจากแบบสอบถาม |

### 1.4 ภาพรวมของระบบ

ระบบนี้เป็นเครื่องมือรายงานโดยผู้ป่วย (Patient Report Outcomes Measurement) ที่พัฒนาขึ้นเพื่อรวบรวมข้อมูลจากผู้ป่วยโรคเบาหวานชนิดที่ 2 โดยมีเป้าหมายเพื่อ:

-   เก็บข้อมูลผลลัพธ์การรักษาจากมุมมองของผู้ป่วยโดยตรง
-   พัฒนาระบบบริการแบบเน้นคุณค่าสำหรับผู้ป่วยเบาหวาน
-   สนับสนุนการวิจัยทางการแพทย์

**พื้นที่ให้บริการ (เขตสุขภาพ):**

1. บริบทคนไทยทีมกลาง (Central)
2. บริบทพื้นที่เพชรบูรณ์ (Phetchabun)
3. บริบทพื้นที่สตูล (Satun)
4. บริบทพื้นที่ลพบุรี (Lopburi)

### 1.5 คำนิยามและคำย่อ

| คำย่อ     | ความหมาย                                            |
| --------- | --------------------------------------------------- |
| PRO       | Patient Reported Outcomes - ผลลัพธ์รายงานโดยผู้ป่วย |
| HbA1c     | น้ำตาลสะสม (Hemoglobin A1c)                         |
| DM Type 2 | โรคเบาหวานชนิดที่ 2 (Diabetes Mellitus Type 2)      |
| Admin     | ผู้ดูแลระบบ (Administrator)                         |

### 3.1 URL สำหรับเข้าใช้งาน

เปิด Web Browser และพิมพ์ URL ของระบบ:

### 3.2 การสมัครสมาชิก (Sign Up)

**ขั้นตอนการสมัครสมาชิก:**

1. เข้าสู่หน้าแรกของระบบ
2. คลิกปุ่ม **"สมัครสมาชิก"** หรือ **"Sign Up"**
3. กรอกข้อมูลตามแบบฟอร์ม:
    - **ชื่อ-นามสกุล** (ชื่อจริง)
    - **อีเมล** (ใช้สำหรับเข้าสู่ระบบ)
    - **รหัสผ่าน** (ความปลอดภัยอย่างน้อย 8 ตัวอักษร)
    - **ยืนยันรหัสผ่าน**
4. คลิกปุ่ม **"สมัครสมาชิก"**
5. ระบบจะนำไปยังหน้าเข้าสู่ระบบโดยอัตโนมัติ

**ข้อกำหนดรหัสผ่าน:**

-   ความยาวอย่างน้อย 8 ตัวอักษร
-   แนะนำให้มีตัวอักษรพิมพ์ใหญ่ พิมพ์เล็ก และตัวเลขผสมกัน

### 3.3 การเข้าสู่ระบบ (Sign In)

**ขั้นตอนการเข้าสู่ระบบ:**

1. เข้าสู่หน้าแรกของระบบ
2. คลิกปุ่ม **"เข้าสู่ระบบ"** หรือ **"Sign In"**
3. กรอกข้อมูล:
    - **อีเมล** ที่ใช้สมัครสมาชิก
    - **รหัสผ่าน**
4. คลิกปุ่ม **"เข้าสู่ระบบ"**
5. หากข้อมูลถูกต้อง ระบบจะนำไปยังหน้าหลัก

### 3.4 การลืมรหัสผ่าน

หากท่านลืมรหัสผ่าน สามารถรีเซ็ตได้ดังนี้:

1. คลิกลิงก์ **"ลืมรหัสผ่าน?"** ในหน้าเข้าสู่ระบบ
2. กรอก **อีเมล** ที่ใช้สมัครสมาชิก
3. คลิกปุ่ม **"ส่งลิงก์รีเซ็ตรหัสผ่าน"**
4. ตรวจสอบอีเมลของท่าน และคลิกลิงก์รีเซ็ตรหัสผ่าน
5. กรอกรหัสผ่านใหม่และยืนยัน
6. เข้าสู่ระบบด้วยรหัสผ่านใหม่

## 4. คู่มือสำหรับผู้ใช้งานทั่วไป

### 4.1 หน้าแนะนำแบบสอบถาม (Introduction)

เมื่อเข้าสู่ระบบแล้ว ท่านจะพบหน้าแนะนำแบบสอบถามซึ่งประกอบด้วย:

-   **วัตถุประสงค์ของการวิจัย:** การวิจัยและพัฒนาเครื่องมือรายงานโดยผู้ป่วย
-   **ประโยชน์:** พัฒนาระบบบริการแบบเน้นคุณค่าสำหรับผู้ป่วยเบาหวาน
-   **การรักษาความลับ:** ข้อมูลทั้งหมดจะปิดเป็นความลับและใช้เฉพาะสำหรับการวิจัยครั้งนี้

**การดำเนินการ:**

-   คลิกปุ่ม **"เริ่มทำแบบสอบถาม"** เพื่อเริ่มต้น

-   **ตัวเลือกหลายข้อ (Checkbox):** คลิกเลือกได้มากกว่าหนึ่งข้อ
-   **ช่องกรอกข้อความ:** พิมพ์ข้อมูลในช่องว่าง

#### 4.2.2 ตอนที่ 2: แบบสอบถามรายงานผลลัพธ์

แบบสอบถามส่วนนี้ใช้ Slider ในการให้คะแนนตามความรู้สึกของท่าน

**วิธีการตอบ:**

1. อ่านคำถามแต่ละข้อ
2. เลื่อน Slider ไปยังตำแหน่งที่ตรงกับความรู้สึกของท่าน
3. **คะแนน 1** = ไม่เห็นด้วยอย่างยิ่ง / น้อยที่สุด
4. **คะแนน 5** = เห็นด้วยอย่างยิ่ง / มากที่สุด

**หมายเหตุสำหรับเขตกลาง (Central):**

-   แบบสอบถามจะแบ่งเป็นขั้นตอนย่อย
-   แสดง Progress Bar แสดงความคืบหน้า
-   หากคะแนนในบางกลุ่มคำถามต่ำ อาจมีคำถามเพิ่มเติม

4. คลิก **"ยืนยัน"** เพื่อส่งแบบสอบถาม
5. รอจนระบบแสดงข้อความ "ส่งข้อมูลสำเร็จ"
6. ระบบจะนำท่านกลับสู่หน้าหลัก

## 5. คู่มือสำหรับผู้ดูแลระบบ

### 5.1 การเข้าสู่หน้า Admin

1. เข้าสู่ระบบด้วยบัญชีที่มีสิทธิ์ผู้ดูแลระบบ
2. ระบบจะแสดงเมนู Admin บน Navigation Bar
3. คลิก **"Admin"** เพื่อเข้าสู่หน้าดูแลระบบ

### 5.2 หน้า Dashboard

หน้า Dashboard แสดงภาพรวมของระบบ:

**ข้อมูลที่แสดง:**

-   **ยอดผู้ตอบแบบสอบถามทั้งหมด:** จำนวนรวมทั้งหมด
-   **แยกรายเขตสุขภาพ:**
    -   บริบทคนไทยทีมกลาง (Central)
    -   บริบทพื้นที่เพชรบูรณ์ (Phetchabun)

### 5.3 การดู Submissions ทั้งหมด

1. คลิกเมนู **"Submissions"** ในแถบเมนูด้านซ้าย
2. ระบบจะแสดงรายการแบบสอบถามทั้งหมดในรูปแบบตาราง

**ข้อมูลที่แสดงในตาราง:**
| คอลัมน์ | คำอธิบาย |
|---------|----------|
| ID | รหัสแบบสอบถาม |
| ชื่อ-นามสกุล | ชื่อผู้ตอบแบบสอบถาม |
| เขตสุขภาพ | พื้นที่ที่ทำแบบสอบถาม |
| วันที่ส่ง | วันและเวลาที่ส่งแบบสอบถาม |
| สถานะ | สถานะของแบบสอบถาม |
| การดำเนินการ | ปุ่มดูรายละเอียด |

**การค้นหาและกรองข้อมูล:**

-   ใช้ช่องค้นหาเพื่อค้นหาตามชื่อหรือ ID
-   ใช้ Dropdown เพื่อกรองตามเขตสุขภาพ

### 5.4 การดูรายงาน (Report) รายบุคคล

1. ในหน้า Submissions คลิกปุ่ม **"ดูรายงาน"** หรือคลิกที่แถวของแบบสอบถามที่ต้องการ
2. ระบบจะแสดงรายละเอียดแบบสอบถามของบุคคลนั้น

**ข้อมูลในรายงาน:**

-   ข้อมูลทั่วไปของผู้ตอบ
-   คำตอบทั้งหมดในแต่ละส่วน
-   คะแนนและการวิเคราะห์

**การพิมพ์รายงาน:**

1. คลิกปุ่ม **"พิมพ์รายงาน"** (Print Report)
2. ระบบจะเปิดหน้าต่างพิมพ์ของ Browser
3. เลือกเครื่องพิมพ์หรือบันทึกเป็น PDF

### 5.5 การ Export ข้อมูล

ระบบรองรับการส่งออกข้อมูลในรูปแบบ Excel

**ขั้นตอน:**

1. ไปที่หน้า Submissions
2. คลิกปุ่ม **"Export Excel"**
3. เลือกเขตสุขภาพที่ต้องการ Export (หรือเลือกทั้งหมด)
4. ระบบจะดาวน์โหลดไฟล์ Excel โดยอัตโนมัติ

**ข้อมูลใน Excel:**

-   ข้อมูลทั่วไปของผู้ตอบทุกคน
-   คำตอบในแต่ละข้อ
-   คะแนนรวมและคะแนนรายหมวด

### 5.6 การจัดการผู้ใช้ในระบบ

1. คลิกเมนู **"Users"** ในแถบเมนู
2. ระบบจะแสดงรายชื่อผู้ใช้ทั้งหมด

**การดำเนินการที่สามารถทำได้:**

-   ดูรายละเอียดผู้ใช้
-   เปลี่ยนสิทธิ์ผู้ใช้
-   ระงับบัญชีผู้ใช้ (ถ้ามี)

## 6. การแก้ไขปัญหาเบื้องต้น

### 6.1 ตารางปัญหาและวิธีแก้ไข

| ปัญหา                        | สาเหตุที่เป็นไปได้          | วิธีแก้ไข                                        |
| ---------------------------- | --------------------------- | ------------------------------------------------ |
| เข้าสู่ระบบไม่ได้            | รหัสผ่านไม่ถูกต้อง          | กดลิงก์ "ลืมรหัสผ่าน" เพื่อรีเซ็ต                |
| หน้าจอค้าง                   | การเชื่อมต่ออินเทอร์เน็ตช้า | Refresh หน้าเว็บ (กด F5) หรือตรวจสอบการเชื่อมต่อ |
| กดปุ่มถัดไปไม่ได้            | ยังกรอกข้อมูลไม่ครบ         | ตรวจสอบและกรอกข้อมูลให้ครบทุกข้อที่บังคับ        |
| ไม่เห็นเมนู Admin            | บัญชีไม่มีสิทธิ์ Admin      | ติดต่อผู้ดูแลระบบเพื่อขอสิทธิ์                   |
| Export ไม่ได้                | ไม่มีข้อมูลในระบบ           | ตรวจสอบว่ามีข้อมูลอยู่หรือไม่                    |
| หน้าเว็บแสดงผิดปกติ          | Browser เวอร์ชันเก่า        | อัปเดต Browser เป็นเวอร์ชันล่าสุด                |
| ไม่ได้รับอีเมลรีเซ็ตรหัสผ่าน | อีเมลไปอยู่ใน Spam/Junk     | ตรวจสอบโฟลเดอร์ Spam                             |

### 6.2 การ Refresh หน้าเว็บ

-   **Windows/Linux:** กด `F5` หรือ `Ctrl + R`
-   **Mac:** กด `Command + R`
-   **Hard Refresh (ล้าง Cache):**
    -   Windows: `Ctrl + Shift + R`
    -   Mac: `Command + Shift + R`

### 6.3 การล้างข้อมูล Browser (Clear Cache)

หากพบปัญหาการแสดงผลผิดปกติ:

1. เปิดเมนูตั้งค่าของ Browser
2. ไปที่ **Privacy & Security** หรือ **ความเป็นส่วนตัวและความปลอดภัย**
3. เลือก **Clear browsing data** หรือ **ล้างข้อมูลการท่องเว็บ**
4. เลือก **Cached images and files**
5. คลิก **Clear data**
6. ลองเข้าใช้งานระบบอีกครั้ง

## 7. คำถามที่พบบ่อย

### Q1: สามารถทำแบบสอบถามได้กี่ครั้ง?

**A:** ผู้ใช้แต่ละคนสามารถทำแบบสอบถามได้หลายครั้ง โดยระบบจะบันทึกประวัติทั้งหมด

### Q2: ข้อมูลของฉันจะถูกเก็บเป็นความลับหรือไม่?

**A:** ใช่ ข้อมูลทั้งหมดจะถูกเก็บเป็นความลับและใช้เพื่อวัตถุประสงค์ในการวิจัยเท่านั้น

### Q3: สามารถแก้ไขแบบสอบถามหลังส่งได้หรือไม่?

**A:** หลังจากส่งแบบสอบถามแล้วไม่สามารถแก้ไขได้ หากต้องการแก้ไข กรุณาติดต่อผู้ดูแลระบบ

### Q4: ใช้เวลานานแค่ไหนในการทำแบบสอบถาม?

**A:** ประมาณ 15-20 นาที ขึ้นอยู่กับความละเอียดในการตอบ

### Q5: รองรับการใช้งานบนมือถือหรือไม่?

**A:** รองรับ ระบบออกแบบมาให้ใช้งานได้ทั้งบนคอมพิวเตอร์ แท็บเล็ต และสมาร์ทโฟน

### Q6: ลืมรหัสผ่านทำอย่างไร?

**A:** คลิกลิงก์ "ลืมรหัสผ่าน" ในหน้าเข้าสู่ระบบ และทำตามขั้นตอนที่ระบบแนะนำ

### Q7: Export ข้อมูลได้ในรูปแบบใดบ้าง?

**A:** ขณะนี้รองรับการ Export ในรูปแบบ Excel (.xlsx)
